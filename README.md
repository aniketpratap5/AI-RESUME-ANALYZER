# AI-RESUME-ANALYZER
Built a smart web application using Python, Flask, HTML, CSS, JavaScript, and Machine Learning that helps students and job seekers analyze their resumes intelligently.  🔍 Key Features: ✅ ATS Score Prediction ✅ Resume Skill Extraction ✅ AI-Based Job &amp; Internship Recommendations ✅ Skill Gap Analysis ✅ Match Percentage for Jobs .
