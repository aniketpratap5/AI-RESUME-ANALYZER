# 🚀 AI Resume Analyzer Pro

AI Resume Analyzer Pro is an intelligent web application that analyzes resumes using Artificial Intelligence and Machine Learning techniques. It extracts skills from resumes, calculates ATS scores, recommends suitable jobs/internships, and identifies missing skills required for specific roles.

---

# ✨ Features

✅ Upload Resume in PDF or DOCX format
✅ Automatic Skill Extraction
✅ ATS Score Calculation
✅ AI-Based Job Matching
✅ Internship Recommendations
✅ Skill Gap Analysis
✅ Match Percentage Visualization
✅ Modern & Attractive User Interface
✅ Responsive Design with Animations

# 🤖 How It Works

1. User uploads resume
2. Resume text is extracted
3. Skills are detected using keyword matching
4. ATS score is calculated
5. Resume skills are compared with job requirements
6. Best matching jobs and internships are displayed

---

# 📊 ATS Score Formula

```python
ATS Score = (Matched Skills / Total Skills) × 100
```
